# თეორია

ბანერის არის სერვერის სახელი და ვერცია რომელიც იწერება სერვერში შესვლის დროს. სერვერის ვერსიის დადგენა საინტერესოა ექსპლოიტების მოსაძებნად.

# ზოგადი პრაქტიკა

## telnet

** ო.ს: იუნიქსი ან ვინდოუსი**

```
telnet IP პორტი
```

ან

```
telnet სერვერის-სახელი პორტი
```

## netcat

**ო.ს: იუნიქსი ან ვინდოუსი**

```
nc -v -n IP პორტი
```

## scanline

**ო.ს: ვინდოუსი**

```
sl -v -b IP
```

## amap

**ო.ს: იუნიქსი**

```
amap -B IP პორტი
```
